#### listing contents  

###### windows  
    dir   
    
###### linux/osx  
    ls -a   
    
#### finding file or program  

###### windows  
    where [whatever]  
    
###### linux/osx   
    which [whatever]  
    
    whereis [whatever]  

###### Pinging a port:

    curl -s IP_ADDRESS:PORT >/dev/null && echo Success. || echo Fail.
