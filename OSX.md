##### Editing or entering a path in Finder

    Command + Shift + G
    
##### Locking the screen

    Control + Shift + Power
    
###### on newer versions it's:
    
    Control + Command + Q 
    
##### Switching workspaces / desktops

    Control + Left/Right
    
##### Scaling text

    System preferences > Display
    
##### Spotlight / Search Everywhere

    Command + Space
